# Beginner ML Project: Student Score Prediction

This is a beginner-friendly machine learning project.

We will predict a student's exam score from the number of hours they studied.

This project is intentionally simple:

- small dataset
- no heavy libraries
- model built from scratch
- easy to read and modify

## What You Will Learn

- what a dataset looks like
- how to split data into training and testing sets
- how linear regression works
- how to make predictions
- how to measure model error

## Example Questions To Explore

- What happens if the data is noisy?
- What happens if study hours and scores are not strongly related?
- Why do we test on data the model did not train on?
